# Progest 要件ドラフト

## 概要
progestはクリエイターのためのプロジェクト管理ソフト。命名規則ファーストな設計と、柔軟な管理方法を強みとして持つ。

## 開発環境
- とりあえずはTauri(Rustを使いたいため)
    - 後々ネイティブアプリケーションにしたい
    - UIはshadcn/uiのコンポーネントで構築
- モノレポ構成
    - crates/ と app/ みたいなふうにしたい

## Mustで欲しい機能
- 命名規則をプロジェクト・ディレクトリ内で設定、強制させれる。
    - snake_caseなどの指定や連番、prefixやsuffix機能も
    - AIによる命名提案機能
- サイドカーファイルによるファイル・ディレクトリ管理
    - Unityのように`.meta`ファイルを作成し、そこにタグなどの情報を含める。元ファイルは改変しない
- CommandPaletteによる柔軟な検索と操作
　  - tag:とかそういう細かい検索をできるように
- ディレクトリ構造や設定を書き出して、チームで共有。新規作成時に参照して同じディレクトリ構造を再現
- ディレクトリの階層構造を意識しなくてもいいフラットビューモード搭載。自分でディレクトリを降って探すのではなくクエリで検索して使うことを第一に考えるモード
